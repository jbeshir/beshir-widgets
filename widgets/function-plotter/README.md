# function-plotter

Plots any expression in `x` over the range `[-10, 10]` using **Preact core** + **Observable Plot** (offline-bundled, no CDN).

The expression is parsed and evaluated by a small in-tree shunting-yard
evaluator (`src/evaluator.ts`) — no `eval`, no `new Function`, no external
math library. Supports `+ - * / ^`, parentheses, `sin cos tan exp log ln sqrt abs`,
and the constants `pi` and `e`.

The chart is rendered with [`@observablehq/plot`](https://observablehq.com/plot/),
mounted imperatively into a Preact-owned ref. It is responsive (a
`ResizeObserver` re-renders on width changes) and respects
`prefers-color-scheme` for light/dark and `prefers-reduced-motion`.

## Local development

```
cd widgets/function-plotter
npm install
npm run dev
```

## Build

```
npm run build
```

Output goes to `dist/`.

## Embed

```html
<iframe src="https://function-plotter.widgets.beshir.org" loading="lazy" style="width:100%;height:480px;border:0"></iframe>
```
