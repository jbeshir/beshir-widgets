# function-plotter

Renders an SVG plot of any expression in `x`.

## Local development

```
cd widgets/function-plotter
npm install
npm run dev
```

## Build

```
npm run build
```

Output goes to `dist/`.

## Embed

```html
<iframe src="https://function-plotter.widgets.beshir.org" loading="lazy" style="width:100%;height:400px;border:0"></iframe>
```

## Note

This widget is the template other widgets copy. New widgets are normally generated by the `build-widget` Claude skill.
